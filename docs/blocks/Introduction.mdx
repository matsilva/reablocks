# Introduction

Blocks are pre-built components that you can use to build your product.
We offer a variety of blocks to help you build your product faster and more efficiently.

We offer the following blocks:

- Foundation
  - Not Found
  - Support
- Administration
  - Forgot Password
  - Login
  - Register
  - MFA
- Authentication
  - Profile
  - Pricing
  - Billing
