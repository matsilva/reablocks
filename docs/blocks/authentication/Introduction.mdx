# Authentication
The authentication blocks are designed to streamline the user authentication process in your applications.
These blocks are essential for securing access and enabling users to interact with areas requiring authentication.
The blocks include Login, Register, and Forgot Password functionalities, each crafted to offer a seamless and
 intuitive user experience. We have the following blocks available for authentication:

- [Forgot Password](?path=/docs/blocks-authentication-forgot-password--docs)
- [Login](?path=/docs/blocks-authentication-login--docs)
- [Register](?path=/docs/blocks-authentication-register--docs)
- [MFA](?path=/docs/blocks-authentication-mfa--docs)
